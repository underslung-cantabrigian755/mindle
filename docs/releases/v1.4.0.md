## 🗂 Tabs

Open multiple Markdown files in one window. Each window keeps its own independent set of tabs — `⌘N` for a new window, `⌘O` (or drag-and-drop) for a new tab in the current one. `⌘W` closes the active tab when more than one is open, otherwise closes the window. Reopening a file already in a tab just switches to it instead of duplicating.

Inactive tabs hold their state in memory (annotations, scrollback view) and rehydrate instantly on switch — no disk re-read.

Thanks to [@agasthik](https://github.com/agasthik) for the nudge — closes [#3](https://github.com/nonatofabio/mindle/issues/3).

## ✨ YAML frontmatter rendered as code

Files like `SKILLS.md` that begin with a `---`-delimited YAML block now render that block as a syntax-highlighted code fence, instead of two horizontal rules wrapping unstyled text.
