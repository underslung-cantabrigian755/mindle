## ⚡ Live reload

Mindle now picks up external edits automatically. Rewrite a file in vim, run a `claude code` task that touches it, sync via Dropbox — Mindle re-renders the moment the writer settles. Bursty writes are debounced (size-stability check), and your scroll position is preserved across the reload.

This is the foundation for v2.0's diff-on-reload experience. Today it just keeps the page in sync; soon it will surface changes as Word-style track-changes you can accept or reject.

## ➗ LaTeX math

KaTeX is now bundled. Inline math (`$a^2 + b^2 = c^2$`) and display blocks (`$$ \int e^{-x^2}\,dx = \sqrt{\pi} $$`) render in-place — no servers, no internet, no font fallbacks. Dollar amounts in prose stay plain text (`$5 and $10` won't trigger math).

## 🔍 Quick Look extension

Hit Spacebar on any `.md` file in Finder and Mindle's full reader pipeline renders the preview — markdown, code highlighting, frontmatter, mermaid, math. Same rendering, no editor window.

## 📎 Make Mindle the default for Markdown

A new menu item under **Mindle → Set as Default for Markdown…** walks you through the Finder Get Info path. If you have a `.md` file open, the dialog offers to reveal it in Finder so you can ⌘I → Open with → Mindle → Change All without hunting for a sample.
